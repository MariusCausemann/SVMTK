# BrainMesh

## Requirements

 - CGAL-4.13 with EIGEN >= 3.2
 - Python3.6
 - python3-dev

## Pybind11

If you did not clone this project using "git clone --recursive", pull pybind11 by
`git submodule update --recursive`

## Installation

run "python setup.py install"

## Docker

There is a Dockerfile in ```docker/`
